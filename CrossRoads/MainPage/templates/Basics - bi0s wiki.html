Comparision operators:¶

  • a == b Equal to condition.
  • a < b Less than condition.
  • a > b Greater than condition.
  • a <= b Less than or equal to condition.
  • a >= b Greater than or equal to condition.
  • a != b Not equal to condition.

Binary Operations¶

To understand this, use python built in funtion bin(n). It will show the binary form of number n.

1
2
3
4
>>> bin(3)
'0b11'
>>> bin(2)
'0b10'

a&b bitwise AND of a,b:¶

1
2
>>> 3&2
2
Each bit is taken taken and AND opeartion is performed. so the result we get is 0b010, which is 2.

a|b bitwise OR of a,b:¶

1
2
>>> 3|2
3
Each bit is taken and OR opeartion is performed. so the result we get is 0b11, which is 3.

a^b bitwise XOR of a,b:¶

1
2
>>> 3^2
1
Each bit is taken and XOR opeartion is performed. In XOR, if both bits are same then the result is 0, else 1. So 1^1 is 0 while 1^0 is 1. Hence, the result we get for 3^2 (0b11^0b10) is 0b001, which is 1.

a<<b Left Shift¶

a<<b it will shift the bits of a in binary format to left, this shift is done b times: 

1
2
3
4
>>> 3<<2
12
>>>bin(12)
0b1100
Binary representation of 3 is 0b11, which is shifted twice to left. So the result is 0b1100 that is 12.

a>>b Right Shift¶

a>>b it will shift the bits of a in binary format to right, this shift is done b times: 

1
2
>>> 3>>1
1
Binary representation of 3 is 0b11, which is shifted once to right. So the result is 0b1 that is 1.

Python can also understand logical operations when written in english : - and - or - not - in
